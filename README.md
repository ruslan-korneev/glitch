# Installation
```bash
git clone git@github.com:ruslan-korneev/gitlab-tg-interface.git
cd gitlab-tg-interface

```

## Locally for development
```bash
python3.11 -m venv .venv
. .venv/bin/activate

poetry install
pre-commit install
```

## Docker
```bash
docker compose up -d --build
```
